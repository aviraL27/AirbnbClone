Airbnb Clone

A static Airbnb homepage clone built using HTML5 and CSS3.
The project focuses on practicing layout structure, Flexbox, spacing, and UI composition without using JavaScript or frameworks.

Features

Navbar with logo, search bar, and icons

Airbnb-style search layout (Where / When / Who)

Property listings grouped by city

Clean card layout with images and pricing

Footer with links and legal section

Tech Stack

HTML5

CSS3 (Flexbox)

Purpose

This project was built to strengthen frontend fundamentals and understand how modern website layouts are structured using only HTML and CSS.
